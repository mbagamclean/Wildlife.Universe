# SEO Stack Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a complete production SEO stack to Wildlife Universe — security headers, robots.txt, sitemap, GA4, rich metadata, JSON-LD schemas, and an RSS feed — so the site ranks well and is fully crawlable from day one.

**Architecture:** All metadata uses Next.js 15 App Router conventions (`export const metadata`, `generateMetadata`, `generateStaticParams`). GA4 is loaded via a dedicated server component injected in `RootLayout`. JSON-LD schemas are inlined as `<script type="application/ld+json">` tags inside each page's `generateMetadata` (via `other`) or rendered as a server component. Content data stays in localStorage (not crawlable), so post/animal/plant/bird metadata is generated from the URL slug alone with sensible defaults — ready to upgrade when a real backend is added.

**Tech Stack:** Next.js 15 App Router, `next/script` (GA4), JSON-LD, Next.js `robots.js` / `sitemap.js` conventions, HTTP headers via `next.config.mjs`.

## Task 3: sitemap.js

**Files:**
- Create: `app/sitemap.js`

- [ ] **Step 1: Create app/sitemap.js**

```js
import { categories, labelSlug } from '@/lib/mock/categories';

const BASE = 'https://www.wildlifeuniverse.org';

export default function sitemap() {
  const now = new Date().toISOString();

  const staticRoutes = [
    { url: BASE, lastModified: now, changeFrequency: 'daily', priority: 1.0 },
    { url: `${BASE}/posts`, lastModified: now, changeFrequency: 'daily', priority: 0.9 },
  ];

  const categoryRoutes = categories.map((cat) => ({
    url: `${BASE}/${cat.slug}`,
    lastModified: now,
    changeFrequency: 'weekly',
    priority: 0.85,
  }));

  const labelRoutes = categories.flatMap((cat) =>
    cat.labels.map((label) => ({
      url: `${BASE}/${cat.slug}/${labelSlug(label)}`,
      lastModified: now,
      changeFrequency: 'weekly',
      priority: 0.7,
    }))
  );

  return [...staticRoutes, ...categoryRoutes, ...labelRoutes];
}
```

```jsx
const GA_ID = 'G-XL797FMXH9';

export function GoogleAnalytics() {
  return (
    <>
      <Script
        src={`https://www.googletagmanager.com/gtag/js?id=${GA_ID}`}
        strategy="afterInteractive"
      />
      <Script id="google-analytics" strategy="afterInteractive">
        {`
          window.dataLayer = window.dataLayer || [];
          function gtag(){dataLayer.push(arguments);}
          gtag('js', new Date());
          gtag('config', '${GA_ID}', {
            page_path: window.location.pathname,
          });
        `}
      </Script>
    </>
  );
}
```

```jsx
const BASE = 'https://www.wildlifeuniverse.org';

function slugToTitle(slug) {
  return slug
    .split('-')
    .map((w) => w.charAt(0).toUpperCase() + w.slice(1))
    .join(' ');
}

export async function generateMetadata({ params }) {
  const { slug } = await params;
  const title = slugToTitle(slug);
  const url = `${BASE}/posts/${slug}`;
```

```js
import { categories } from '@/lib/mock/categories';

const BASE = 'https://www.wildlifeuniverse.org';
const BUILD_DATE = new Date().toUTCString();

export async function GET() {
  const categoryItems = categories
    .map(
      (cat) => `
    <item>
      <title>${cat.name} — Wildlife Universe</title>
      <link>${BASE}/${cat.slug}</link>
      <description>Explore ${cat.name.toLowerCase()} species and stories on Wildlife Universe.</description>
      <pubDate>${BUILD_DATE}</pubDate>
      <guid isPermaLink="true">${BASE}/${cat.slug}</guid>
    </item>`
    )
    .join('');

  const rss = `<?xml version="1.0" encoding="UTF-8"?>
<rss version="2.0" xmlns:atom="http://www.w3.org/2005/Atom">
  <channel>
    <title>Wildlife Universe</title>
    <link>${BASE}</link>
    <description>Discover remarkable species, conservation milestones, and the wonders of the natural world.</description>
    <language>en-us</language>
    <lastBuildDate>${BUILD_DATE}</lastBuildDate>
    <atom:link href="${BASE}/rss" rel="self" type="application/rss+xml"/>
    <image>
      <url>${BASE}/favicon.png</url>
      <title>Wildlife Universe</title>
      <link>${BASE}</link>
    </image>
    ${categoryItems}
  </channel>
</rss>`;

  return new Response(rss, {
    headers: {
      'Content-Type': 'application/rss+xml; charset=utf-8',
      'Cache-Control': 'public, max-age=3600, s-maxage=3600',
    },
  });
}
```
